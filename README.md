# 📱 Social Media Platform

A modern mini Social Media Platform built using Django, HTML, CSS, JavaScript, and SQLite.

---

## 🚀 Features

- User Registration & Login
- User Profiles
- Create Posts
- Upload Images
- Like & Comment System
- Follow / Unfollow Users
- Search Users
- Dark Mode
- Responsive UI
